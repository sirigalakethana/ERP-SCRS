# 📘 ERP-Based Student Course Registration System (ERP-SCRS)

## 📌 Project Description
This project is a Python-based mini ERP (Enterprise Resource Planning) system designed to manage student course registration in universities. It automates the process of course selection, ensuring prerequisite validation, seat availability, and timetable clash detection. The system also provides basic analytics to understand course enrollment trends.

---

## 🎯 Objective
To design and develop an ERP-based system that simplifies and automates student course registration by handling academic constraints like prerequisites, seat limits, and scheduling conflicts efficiently.

---

## 🚀 Features
- Student Profile Management
- Course Catalog Viewing
- Course Registration (Add Course)
- Course Drop Functionality
- Prerequisite Validation
- Seat Availability Check
- Timetable Clash Detection
- Enrollment Analytics (Graph Visualization)
- CSV-based Data Handling

---

## 📌 Use Cases
- Student registers for a course
- System checks prerequisite completion
- System verifies seat availability
- System detects timetable clashes
- Student drops a course
- System generates analytics for course enrollment
